# SmartCalendar

## Опис проєкту
SmartCalendar — це настільний додаток на Python з використанням Tkinter, який дозволяє:

* Переглядати календар у стандартному місячному вигляді;
* Додавати, редагувати та видаляти події;
* Підсвічувати поточний день;
* Позначати дні, в яких є події;
* Зберігати події у форматі JSON, щоб вони залишались після закриття програми.

Проєкт побудований модульно, що дозволяє робити коміти по етапах розробки та відстежувати їх на Kanban-дошці.

---

## Структура проєкту

```
calendar_project/
│
├─ main.py             # запуск програми
├─ models.py           # класи: User, Event, CalendarModel
├─ storage.py          # збереження та завантаження подій
├─ calendar_gui.py     # GUI (Tkinter) з управлінням подіями
└─ events.json         # файл збережених подій (створюється після першого запуску)
```

---

## Встановлення та запуск

1. Клонування репозиторію:

```bash
git clone <URL-репозиторію>
cd calendar_project
```

2. Встановлення Python (якщо ще не встановлений):

* Потрібен Python 3.10+

3. Запуск додатку:

```bash
python main.py
```

---

## Використання

* Кнопки `<` та `>` для переходу між місяцями.
* Клік на день відкриває список подій і кнопку "Додати подію".
* Натискання на подію показує її деталі з можливістю редагування або видалення.
* Події автоматично зберігаються у `events.json`.

## Kanban

Кожен етап відповідає окремій картці на Kanban-дошці:

* **To Do**: планування функцій календаря та подій
* **In Progress**: розробка модулів по черзі
* **Done**: інтеграція та тестування GUI, збереження подій
